# 📄 Requisitos do Sistema - Atividades Acadêmicas Complementares

## ✅ Requisitos Funcionais (RF)

- RF01: O sistema deve permitir que o aluno faça login.
- RF02: O sistema deve permitir cadastrar atividades acadêmicas complementares.
- RF03: O sistema deve permitir anexar comprovantes (PDF, imagem).
- RF04: O sistema deve permitir visualizar atividades cadastradas.
- RF05: O sistema deve permitir editar ou excluir atividades.
- RF06: O sistema deve permitir que o professor/coordenação valide atividades.
- RF07: O sistema deve calcular a carga horária total do aluno.
- RF08: O sistema deve exibir o status da atividade (pendente, aprovado, reprovado).

---

## ⚙️ Requisitos Não Funcionais (RNF)

- RNF01: O sistema deve ser responsivo (funcionar em celular e computador).
- RNF02: O sistema deve garantir segurança no login (autenticação).
- RNF03: O sistema deve permitir upload de arquivos com rapidez.
- RNF04: O sistema deve estar disponível 24 horas por dia.
- RNF05: O sistema deve ter tempo de resposta inferior a 2 segundos.
